# Dataset de países - Gestión
## Descripción
Este proyecto es una aplicación en Python que permite gestionar información de países a partir de un archivo CSV. 
Permite realizar búsquedas, filtros, ordenamientos y generar estadísticas básicas sobre población y superficie.
Forma parte del Trabajo Práctico Integrador (TPI) de la materia **Programación 1** de la Tecnicatura Universitaria en Programación.

## Funcionalidades principales
- Buscar países por nombre (coincidencia parcial o exacta)
- Filtrar por continente, rango de población o superficie
- Ordenar por nombre, población o superficie
- Mostrar estadísticas:
  - País con mayor y menor población
  - Promedio de población y superficie
  - Cantidad de países por continente

## Tecnologías utilizadas
- Python 3.x
- Módulos estándar:
  - `csv` (lectura de archivos)
  - `statistics` (promedios)

## Instrucciones de uso
1. Clonar el repositorio:
   ```bash
   git clone https://github.com/xogxsss/trabajo_integrador.git
2. Abrir el proyecto en un entorno con Python 3 instalado.
3. Asegurarse de que el archivo dataset_paises.csv esté en la misma carpeta que el código principal.
4. Ejecutar el programa:
     `python main.py`
5. Seguir el menú interactivo desde la consola.

## Ejemplo de ejecución
```
------ MENÚ DE OPCIONES ------
1. Filtrar países
2. Ordenar países
3. Mostrar estadísticas
4. Buscar país por nombre
5. Salir

```
